# Sinfim.uz / Online School Platform - Product Requirements Document ## Overview A multi-tenant LMS SaaS platform designed for schools, education centers, and individual teachers to transition from manual operations (like Telegram) to a structured web workspace. ## Core Principles - **Desktop-First:** 1440px wide layouts, optimized for Chrome and web browsers. - **Operational Focus:** Not a marketplace, but a management tool for school owners, mentors, and students. - **Multi-Tenant:** Each school has its own branded workspace (e.g., `sinfim.uz/school-slug`). ## Domain Model - **Course:** Reusable content (Lessons, Videos, Materials, Quizzes). - **Class / Group:** Live operations (Students, Mentors, Start Dates, Schedules). ## Visual Direction - **Vibe:** Trustworthy, organized, modern, clean, calm. - **Palette:** Balanced, professional (Avoid heavy gradients, purple/blue startup looks, or overly playful visuals). - **Layout:** Wide content areas, desktop sidebars, data tables, and structured spacing. ## Target Screens (16 Total) 1. Platform Landing Page 2. Entry Point Page 3. School Request Page 4. Superadmin Organization Create 5. Owner Dashboard (SaaS Wide View) 6. Course Detail (Tabs & Tables) 7. Class / Group Detail 8. Lesson Editor 9. Homework Review Inbox (Split-Pane) 10. Student Dashboard 11. Student Lesson Detail 12. Organization Settings 13. Public School Page 14. Public Course Page 15. Demo School 16. Login / First Password Flow
